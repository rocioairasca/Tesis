# GrowSync

Proyecto de tesis: GrowSync — aplicación para gestión agropecuaria.

Resumen rápido:
- Frontend: `grow-sync/` (React, Ant Design)
- Backend: `growsync-backend/` (Express, Supabase, cron jobs)

Comenzar (desarrollo):

- Frontend:
	- Abrir `grow-sync/`
	- `npm install` y `npm start`
- Backend:
	- Abrir `growsync-backend/`
	- `npm install` y `node index.js` (o configurar `npm start`)

Documentación en `docs/`:
- `docs/frontend/*`: descripción, setup y arquitectura del frontend
- `docs/backend/`: endpoints y guías de backend (pendiente de completar)

Notas importantes:
- Hay archivos `.env` con ejemplos en los subproyectos; sustituir valores sensibles antes de publicar.
- Añadir `.env.example` y no subir secretos a git.

Ver también: `grow-sync/README.md` y `growsync-backend/README.md` (nuevo)
